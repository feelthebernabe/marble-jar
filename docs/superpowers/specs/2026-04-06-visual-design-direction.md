# Marble Jar — Visual Design Direction

## Aesthetic: Punk Zine

Not a "designer's interpretation of a zine." Actual 1970s UK punk fanzine energy — Sniffin' Glue, Chainsaw, Toxic Graffiti. Photocopied, chaotic, layered, raw.

Reference: punk zine collage image provided by Michelle (pink/black/white xeroxed fanzines piled on a table).

## Core Palette

| Token | Value | Usage |
|-------|-------|-------|
| `--bg-paper` | `#f0e6d3` | Kraft paper background |
| `--ink` | `#000000` | Borders, text, drop shadows |
| `--accent` | `#e8175d` | Hot pink — highlights, stamps, tape strips, active states |
| `--white` | `#ffffff` | Card/scrap backgrounds |
| `--marble-*` | per-user | Saturated flat fills — no gradients, no 3D |

Everything else derives from these five. No soft grays, no jewel tones, no warm cream.

## Typography

Mixed fonts — ransom-note energy. No single typeface across the app.

| Font | Role | Vibe |
|------|------|------|
| **Permanent Marker** | App title, big headers | Hand-scrawled, edgy, legible |
| **Archivo Black** | Labels, stamps, nav, section headers | Bold blocky, poster type |
| **Special Elite** | Body text, subtitles, descriptions | Typewriter, slightly worn |
| **Courier Prime** | Stats, data, streaks, timestamps | Monospace, utilitarian |

Rules:
- Never use one font for an entire screen
- Headers should feel hand-written or stamped, not typeset
- Body text is typewriter, not sans-serif
- Stats/numbers are monospace

## Visual Elements

### Borders & Shadows
- **3-4px solid black borders** on all cards/containers
- **Hard drop shadows**: `box-shadow: 4-5px 4-5px 0 #000` — no blur, no soft shadows ever
- No rounded corners beyond 4px (except marbles which are circles)

### Textures
- Xerox/photocopy grain overlay on the page (`mix-blend-mode: multiply`, subtle SVG noise)
- Lined paper background on index-card style components (streaks, settings)
- Dashed borders for dividers (not solid hairlines)

### Layout Chaos
- Cards and scraps rotated slightly (`transform: rotate(0.5deg)` to `rotate(-1.2deg)`)
- Nothing perfectly aligned — alternate odd/even rotations on lists
- Tape strips at angles on key elements
- Hand-written annotations in pink (Permanent Marker, small size)

### Marbles
- Flat circles with **3px black border**
- Solid saturated fill — no gradients, no reflections, no 3D
- Symbol stamped in center (unicode: star, moon, heart, lightning, diamond, etc.)
- Each marble rotated slightly at a random angle inside the jar
- Halftone dot overlay for photocopied texture feel

### The Jar
- Simple rectangle with thick black border and slight border-radius at bottom
- Not glass, not transparent — it's a **drawn jar** with a solid black lid
- Flat kraft/off-white interior
- Marbles pile up at chaotic angles, not neatly stacked

### Stamps
- Inline labels like streak counts, status badges
- Border + text in accent pink or black
- Rotated a few degrees, uppercase, letterspaced
- Think: rubber stamp, not pill badge

## Animation Direction

Animations should feel like **things being stamped, slapped, or pasted onto a page** — not physics simulations.

| Action | Animation |
|--------|-----------|
| Marble earned | Stamp/slap down with slight overshoot and rotation snap |
| Feed item appears | Slides in and sticks, like a scrap being pasted |
| Screen transition | Paper shuffle / stack reorder |
| Streak dot fills | Quick ink-fill, not smooth fade |
| Celebration | Full-page confetti of stamps, stickers, marbles — chaotic |

No spring physics. No smooth easing. Snappy, immediate, slightly rough.

## Navigation

- Bottom nav bar: solid black background, hot pink active state
- Font: Permanent Marker, all caps
- Three items: JAR / FEED / PROFILE
- Hard top border in accent pink

## What This Is NOT

- Not warm/cozy/kitchen-counter skeuomorphism
- Not clean minimalism with Inter and gray-50 backgrounds
- Not "cute zine" with washi tape and neat torn edges
- Not dark mode productivity app
- Not anything with subtle shadows, soft gradients, or rounded sans-serif type
- Not anything that looks like it was generated by an AI
